# Cynergy Bank — Open Banking (OBIE Read/Write)

> Cynergy Bank is an FCA/PRA-authorised UK specialist bank (FCA ref 575105) and a PSD2 ASPSP. It exposes the UK Open Banking (OBIE) Read/Write API families — Account & Transaction Information (AIS), Payment Initiation (PIS), and Confirmation of Funds (CBPII) — to FCA/EEA-regulated Third Party Providers. Access is FAPI 1.0 Advanced: OAuth2/OIDC with PSD2 strong customer authentication, mutual-TLS client authentication, and detached-JWS request signing using OBIE/eIDAS certificates. The OpenAPI documents are the shared OBIE Read/Write standard (v4.0.1) the interface conforms to, not a Cynergy-proprietary contract. The production endpoints are OBIE-directory registered and mTLS/eIDAS-gated; there is no anonymously reachable API host, and as a non-CMA9 bank Cynergy publishes no CMA Open Data reference API.

## APIs
- [Account & Transaction Information API (AIS)](https://developer.openbanking.cynergybank.co.uk/): 29 operations — account access consents, accounts, balances, transactions, beneficiaries, direct debits, standing orders, scheduled payments, statements, parties, products, offers.
- [Payment Initiation API (PIS)](https://developer.openbanking.cynergybank.co.uk/): 41 operations — domestic, domestic-scheduled, domestic-standing-order, international, international-scheduled, international-standing-order, and file payments, each with consent + funds-confirmation + submission.
- [Confirmation of Funds API (CBPII)](https://developer.openbanking.cynergybank.co.uk/): 4 operations — funds-confirmation consents and point-in-time funds confirmations.
